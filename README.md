HTML Presentation Framework
---------------------------

**About this repo**

+ A simple HTML presentation framework. Currently only Chrome and Safari are supported.


**Read the blog post**

+ http://forcefront.com/2012/html-presentation-framework/


**Try out the demo**

+ http://myst729.github.com/Presentation


**How to use**

+ Back  : `←`, `↑` or `PgUp`.
+ Next  : `→`, `↓`, `PgDn` or `　　Space　　`.
+ First : `Home`.
+ Last  : `End`.
